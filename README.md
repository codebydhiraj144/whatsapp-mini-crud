# whatsapp-mini-crud
This project is a mini WhatsApp-like chat application developed using Node.js, Express, MongoDB (Mongoose), and EJS. It supports creating, viewing, editing, and deleting chats using RESTful routes and method-override. Built for learning and practice purposes.
